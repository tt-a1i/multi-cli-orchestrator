# v0.1.2 - 分发与安装

日期: 2026-02-26

## 主要更新

- 新增 Python 打包元数据（`pyproject.toml`）和 `mco` 控制台入口。
- 新增 npm 包装器（`@multi-cli-orchestrator/cli`），方便 Node 生态调用。
- 新增 PyPI / npm 发布工作流。
- README 新增安装说明（`pipx`、源码安装、npm 包装器）。

## 安装方式

推荐方式：

```bash
pipx install multi-cli-orchestrator
```

源码安装：

```bash
python3 -m pip install -e .
```

Node 包装器：

```bash
npm i -g @multi-cli-orchestrator/cli
```

## 说明

- npm 包装器依赖系统可执行 `python3`。
- YAML 配置仍需可选依赖：`pip install pyyaml`。
